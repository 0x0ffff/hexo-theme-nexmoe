## 评论支持
- GitTalk
- Gitment
- Valine

如果以上没有满足你的评论框，可以在本主题目录 `layout/_partial/_comment` 新建一个文件 `xxx.ejs`，然后在该文件中安装你需要的评论框系统代码。
如果有空的话，还可以将你的修改 `pull request` 到本项目的 Github

## 国际化
目前中文翻译较全，其他语言翻译不完全，有余力的大佬可以来贡献一下
语言文件在 languages 里，参考 `zh-CN.yml` 进行翻译即可
